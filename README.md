# plan7

基于tcp协议用Go语言编写的聊天程序

安装使用说明
--------------------------------------------------------------------------

安装Redis客户端

go get github.com/go-redis/redis

安装msyql客户端

go get github.com/go-sql-driver/mysql

安装uuid拓展包

github.com/pborman/uuid
